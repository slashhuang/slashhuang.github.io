to be a 好厨师

By slashhuang

Published 2013-07-10

好好学厨艺

现在每个周末，几乎都会自己下厨做点菜，早上就自己烧白粥吃。这样的生活，不仅省钱还很健康，于是，我决定要好好练习厨艺。

失败案例

1.小鸡炖蘑菇(16.7.10)

  • 失败结果： 做成了清汤鸡块炖蘑菇
  • 原因分析：
    1. 鸡块只是粗略的沸水去渣，时长不够。 合理的时长在90秒左右。
    2. 水加的过多。 合理的水应该刚好没过鸡块和蘑菇

  • PC: webpack + jquery + 后端静态资源 + react
  • 移动端: VUe + webpack
