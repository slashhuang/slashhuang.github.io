### 添加Docs，让它成为技术栈专家

每个代码仓库都有明确的技术选型，比如我喜欢的技术栈包括pnpm、vue.js、typescript等，我希望cursor以技术栈专家的身份回答我的问题。

需要`cmd + >` 打开cursor settings，在Features Tab的Docs目录添加技术栈文档
